# 🎓 CampusCore - Student Management System

CampusCore is a **Console-Based Student Management System** developed in **C++**. It helps manage student records efficiently using file handling and standard data structures.

This project demonstrates my understanding of **C++ programming, file handling, searching, sorting, vectors, and menu-driven applications**.

---

## 📌 Features

- ➕ Add New Student
- ❌ Delete Student
- ✏️ Update Student Details
- 🔍 Search Student by Roll Number
- 🔍 Search Student by Name
- 📋 View All Students
- 🔤 Sort Students by Name
- 📊 Sort Students by Marks
- 🏆 Generate Student Report
- 💾 Save Data to File
- 📂 Load Data from File

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/CampusCore.git
```

2. Open the project in VS Code or CodeBlocks.

3. Compile

```bash
g++ main.cpp -o CampusCore
```

4. Run

```bash
./CampusCore
```

For Windows

```bash
CampusCore.exe
```
